# Witchesbrew for [Yazi](https://github.com/sxyazi/yazi)

A dark, muted theme with pink and wine tones — ported from [witchesbrew.nvim](https://github.com/shurjo/witchesbrew.nvim).

## Preview

![preview](preview.png)

## Installation

Copy or symlink `witchesbrew.yazi/` into `~/.config/yazi/flavors/`.

## Usage

Set your `~/.config/yazi/theme.toml`:

```toml
[flavor]
dark = "witchesbrew"
```
